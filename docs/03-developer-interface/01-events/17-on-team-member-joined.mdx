# `onTeamMemberJoined`

Triggered when a new user joins a team where the App is registered.

This is a team-level event, it is not tied to a specific conversation. Use [`onUserJoinedConversation`](15-on-user-joined-conversation.mdx) when you need to react to members being added to a conversation.

## Received properties

- **`userId`** — Qualified identifier of the user who joined the team.
- **`teamId`** — Identifier of the team the user joined, provided as a `TeamId`.
