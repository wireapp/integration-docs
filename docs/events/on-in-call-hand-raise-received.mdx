# `onInCallHandRaiseReceived`

Received the Hand Raise emoji while in a call.

:::warning

Apps can’t join calls right now, so this event won’t fire.

:::

## Received properties

Apart from the [Base properties](../core-concepts/message-model.md#base-message-properties),
a `WireMessage.InCallHandRaise` object includes a boolean.  
When someone raises a hand, `isHandUp` is true. When they lower it, it’s false.
