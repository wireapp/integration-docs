The App ID is required for the following steps.

## On desktop (macOS, Windows, or on Wire for web)

In the app:

1. Select *Settings*, then select *Manage team* or go to [teams.wire.com](https://teams.wire.com/).
2. Log in with your account credentials:

![TM_log_in.png](/img/team-management/log_in.png)

3. Select *Integrations*.

![TM_empty_integrations_tab.png](/img/team-management/empty_integrations_tab.png)

4. Select *Add App*.

![add external app](/img/team-management/add_external_app.png)

5. Enter the app ID.

![external app details](/img/team-management/external_app_details.png)

6. Review the app details.

:::tip
Only add apps from trusted sources.
:::
